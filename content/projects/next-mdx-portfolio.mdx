This is a blog website built with NextJs and Strapi. 

## Features

- **Responsive Design**:The blog website is fully responsive and functions seamlessly across all devices.
## Technologies

- **NextJs**: A React framework for creating both static and dynamic websites.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Strapi**: An open-source headless CMS that enables developers to build and manage customizable APIs with ease.
